Para mostrar o poder do CSS

main > p {
	background: red;
}
> Para selecionar os filhos diretos. Neste caso, todos os "p" filhos diretos do main serão selecionados.

img + p {
	background: blue;
}
+ Para selecionar o irmão mais próximo. Neste caso, apenas o "p" logo após a imagem será selecionado.

img ~ p {
	background: yellow;
}
~ Para selecionar todos os irmãos que vem após a imagem. 
Neste caso, todos os "p" após a imagem serão selecionados.
Note que, como esse comando vem após o anterior e possui o mesmo peso, o primeiro paragrafo não terá mais 
a cor azul e sim a cor amarela, pois o último comando sobrepoem os anteriores, se tiverem mesmo peso.

.principal p:not(#missao) {
	background: orange;
}
:not() Para excluir algum elemento especifico. Neste caso, ele irá excluir o código com id="missao" 
do comando. Assim, enquanto todos os outros paragrafos ficam laranja, o paragrafo da missao ficará
amarelo, pois é o comando anterior a esse.
